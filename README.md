# Diabetic Retinopathy Detection

## Overview

A deep learning-based system that detects diabetic retinopathy severity from retinal fundus images using EfficientNetB0 and transfer learning.

## Features

- Multi-class classification
- Transfer Learning with EfficientNetB0
- Medical image analysis
- Diabetic Retinopathy detection
- Streamlit deployment

## Dataset

APTOS 2019 Blindness Detection Dataset

## Model

EfficientNetB0

## Results

- Validation Accuracy: 71.39%
- Weighted F1 Score: 0.71

## Technologies Used

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
